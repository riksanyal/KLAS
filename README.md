# KLAS
This is the official repository of our paper: [**KLAS: Using Similarity to Stitch Neural Networks for Improved Accuracy-Efficiency Tradeoffs​**](https://arxiv.org/pdf/2605.29259).
KLAS is a framework that uses KL divergence to automatically and efficiently select optimal block-stitching configurations across pretrained neural networks, significantly improving their accuracy-efficiency tradeoffs.
The source code will be released soon.
